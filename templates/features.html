Sentiment and Emotion Detector - Features

Text scanner

Text scanner gets an input as a text (can be a single text or a multi-line text like a poem) and gets the average sentiment and emotion of it

Single-text: gets the text to perform an analisis of the sentiment and the emotions in it

Multi-text: Splits the entire text (using as a separator the ";" character) in each individual text (or verse) and gets the median sentiment and emotion of all texts

Inputs:
  • - Text
  • - Mode
Outputs:
  • - Text
  • - Sentiment
  • - Emotion

Twitter scanner

Twitter scanner makes a request to twitter with the user and the time span specified by the user and gets the count of tweets of every sentiment

Inputs:
  • - User name
  • - Since
  • - Until
Outputs:
  • - Positive tweets
  • - Neutral tweets
  • - Negative tweets

Youtube scanner

Youtube scanner goes to a specified video and gets a specified ammount of comments, then it gets the median emotions of the comments and the count of comments of each sentiment (similar to tweet scanner)

Inputs:
  • - Video link
  • - Max comments
Outputs:
  • - Positive comments
  • - Neutral comments
  • - Negative comments
  • - Emotion percentages

File scanner

File scanner gets a file as input and converts it to text for doing an analisis of emotions and sentiments. The input can be a single-text file or a multi-text file

Single-text: gets the text to perform an analisis of the sentiment and the emotions in it

Multi-text: Splits the entire text (using as a separator the ";" character) in each individual text and gets the median sentiment and emotion of all texts

Inputs:
  • - File
  • - Mode
Outputs:
  • - File sentiment
  • - File emotion
  • - Emotion percentages
